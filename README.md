# Могилёвская Любительская Лига — Волейбол

Веб-приложение для отслеживания расписания, результатов и турнирной таблицы Могилёвской любительской волейбольной лиги.

**Сезон 2025/26** · 8 команд · 14 туров · Два круга · 24 фев 2026 – 6 июн 2026 (включая плей-офф)

## Страницы

| Путь | Описание |
|---|---|
| `/` | Турнирная таблица с очками, партиями, подсветкой лидера и плей-офф зоны |
| `/calendar` | Полный календарь сезона: матчи в сетке по месяцам, клик на матч — карточка с деталями |
| `/tours` | Расписание матчей по турам с датами, залами и временем |
| `/teams` | Список команд с краткой статистикой |
| `/teams/:id` | Страница команды: статистика, форма, все матчи с фильтром Дома/В гостях |
| `/playground` | Интерактивная игра 2048 в волейбольной теме |

## Обновление данных

Данные разделены на три файла:

| Файл | Содержимое |
|---|---|
| `src/data/schedule.ts` | Расписание: кто с кем играет, когда и где. Меняется редко. |
| `src/data/results.ts` | Результаты матчей. **Обновлять после каждого тура.** |
| `src/data/league.ts` | Команды и сборка данных. Трогать не нужно. |

**Добавить результат матча** — открыть `src/data/results.ts` и добавить запись:

```ts
"GW{тур}: {хозяева} - {гости}": {
  setsHome: 3,
  setsAway: 0,
  setScores: [
    { home: 25, away: 20 },
    { home: 25, away: 18 },
    { home: 25, away: 22 },
  ],
},
```

Ключ должен точно совпадать с именами команд из `schedule.ts`. Матч автоматически помечается как сыгранный при наличии записи в `matchResults`.

**Формат:** всегда ровно 3 партии. Возможные счёты: 3:0 или 2:1 (и зеркально 0:3 / 1:2).

**Система начисления очков:** победа 3:0 — 3 очка; победа 2:1 — 2 очка, проигравший получает 1 очко.
